Health Event

      

        Enables FPS + memory sampling via requestAnimationFrame.
        Must be enabled before events are sent — call Enable then Initialize SDK.
        A health event is automatically queued at each session end.
      

      

        
        
        
        
      

      

    


    

      
SDK Init Event

      

        Sent automatically on initialize() when health tracking is enabled.
        Captures SDK version, engine version, build, and first-session flag.
        Use Send Now to fire it manually (health must be enabled).

Health Tracking (P5)

      

        rAF-based FPS sampling + memory info. Enable before or after init.
        See the dedicated Health & Init panel for full controls.
      

      

        
        
        
      

    


    

      
Storage Adapter (P12)

      

        Swap the storage backend. Must be called before initialize().
        The InMemoryAdapter keeps everything in RAM (lost on reload — useful for SSR/Node).
      

      

        
        
      

    


    

      
Custom HTTP Handler (P9)

      

        Intercept every outgoing HTTP request. The handler receives (url, payload, authHeader, callback(status, body)).
        Must be set before initialize().
      

      

        
        
      

    


    

      
External User ID (P7)

      

        Attach your own user identifier (≤64 chars) that gets injected into every event annotation.
        Must be set before initialize().
      

      

        

          
          
        

      

      

        
      

    


    

      
Backoff Interval (P1)

      

        Set the event-queue poll interval. The SDK doubles it automatically on network failures
        (up to 300 s) and resets on success or when the browser comes back online.
      

      

        

          
          
        

      

      

        
      

    


    

      
ESM Import (P2)

      

        The dist/GameAnalytics.esm.js bundle exports:

        import GameAnalytics from '../dist/GameAnalytics.esm.js';

        import { gameanalytics } from '../dist/GameAnalytics.esm.js';


        Open DevTools → Console and run:

        import('../dist/GameAnalytics.esm.js').then(m => console.log(Object.keys(m)))
